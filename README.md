## Kitabisa clone frontend
[Live Demo](https://campaign-fe.vercel.app)

## Prerequisites
Using nextjs and my own starter [Repo](https://github.com/coy102/next-ts-starter)

- [Yarn](https://yarnpkg.com/en/docs/install) v1.x.x

## Quick Start

### Locally

```
yarn install
cp .env.example .env
yarn start
```

### Testing

```
yarn test
yart test:coverage
```
